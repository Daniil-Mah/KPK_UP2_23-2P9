### Вариант №2. Сервис профилей.
#### Добавить профиль.

Информация требуемая для создания профиля:
| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| ID роли | Обязательно | Целое число |  |  |
| ID профиля | Обязательно | Целое число |  |  |
| Фамилия | Обязательно | Строка |  |  |
| Имя | Обязательно | Строка |  |  |
| Отчество | Не обязательно | Строка |  |  |
| Номер телефона | Обязательно | Цело число | равно 11 |  |
| Электронная почта | Обязательно | Строка |  |  |
| Фотография | Обязательно | PNG |  |  |

Комбинация параметров: 'ID роли' и 'ID профиля' должна быть уникальной.

При удачном добавлении профиля вернуть:
| Параметр | Тип |
|---|---|
| ID страницы профиля | Целое число |
| Фамилия | Строка |
| Имя | Строка |
| Отчество | Строка |
| Номер телефона | Цело число |
| Электронная почта | Строка |
| Фотография | PNG |

#### Изменить профиль по ID.

Входные параметры:
| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| ID страницы профиля | Обязательно | Целое число |  |  |
| Фамилия | Не обязательно | Строка |  | NULL |
| Имя | Не обязательно | Строка |  | NULL |
| Отчество | Не обязательно | Строка |  | NULL |
| Номер телефона | Не обязательно | Цело число | равно 11 | NULL |
| Электронная почта | Не обязательно | Строка |  | NULL |
| Фотография | Не обязательно | PNG |  | NULL |

Выходные параметры:
| Параметр | Тип |
|---|---|
| ID роли | Целое число |
| ID профиля | Целое число |
| Фамилия | Строка |
| Имя | Строка |
| Отчество | Строка |
| Номер телефона | Цело число |
| Электронная почта | Строка |
| Фотография | PNG |

#### Удаление группы по ID.
Вернет истинну (`True`) если профиль был закрыт, иначе вернет ложность (`False`)

#### Получить информацию о профиле по ID.
Информация возвращаемая в случае удачного поиска профиля по ID:
| Параметр | Тип |
|---|---|
| ID роли | Целое число |
| ID профиля | Целое число |
| Фамилия | Строка |
| Имя | Строка |
| Отчество | Строка |
| Номер телефона | Цело число |
| Электронная почта | Строка |
| Фотография | PNG |
| Активность профиля | Буллевое значение |
